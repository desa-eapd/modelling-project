**When:** 2023–2026

**Purpose:** Provide a Malaysia-specific OG-Core implementation for dynamic general equilibrium analysis of fiscal policy, including how fiscal reforms affect GDP, investment, consumption, wages, interest rates, and tax revenues.

**What happened:** The public repository runs a baseline and a reform simulation. The reform example changes the **corporate income tax rate to 35%**, solves the transition path, and saves summary percentage changes for macroeconomic aggregates over the first ten years and in the steady state.

**Key insights:** The Malaysia implementation provides the basic machinery for comparing baseline and reform paths for macroeconomic outcomes. Key graphs generated by the repository example cover **GDP**, **consumption**, **capital**, **labor**, **interest rates**, and **wages**. The repository notes that documentation for the specific Malaysian calibration will be available soon, so public results remain limited to the example-run outputs and model structure.

**Resources:** [Repository](https://github.com/Revenue-Academy/OG-MYS). Country-specific documentation is noted in the repository as forthcoming.

**Models:** OG-Core
